Binary Tree Traversal (การท่องต้นไม้)

แนวคิด

การ ท่องต้นไม้ (traversal) คือการเยี่ยมทุกโหนดอย่างเป็นระบบ มี 4 แบบหลัก:

  • Preorder (Root → Left → Right): เยี่ยมพ่อก่อนลูก
  • Inorder (Left → Root → Right): บน BST จะได้ค่าเรียงจากน้อยไปมาก
  • Postorder (Left → Right → Root): เยี่ยมลูกให้ครบก่อนพ่อ
  • Level-order (BFS): เยี่ยมทีละชั้นจากบนลงล่าง ใช้ คิว (queue)

💡 3 แบบแรกเป็น recursive — ดู Call Stack ประกอบได้ · ตัวอย่างนี้สร้างต้นไม้แบบ BST จากค่าที่ใส่

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: มีต้นไม้นิพจน์ (3 + 4) * 5 เก็บเป็น binary tree จงพิมพ์ค่าทุกโหนดและคำนวณผลลัพธ์

แก้ด้วย Binary Tree Traversal อย่างไร: ใช้ postorder (Left → Right → Root) ประเมินลูกทั้งสองก่อนแล้วค่อยประเมินตัวดำเนินการ — ได้ 3,4 ก่อน บวกเป็น 7 แล้วคูณ 5 = 35 · เยี่ยมทุกโหนดครั้งเดียว O(n), หน่วยความจำ O(h) ตามความสูง (call stack)
