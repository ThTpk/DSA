Coin Change — DP (เหรียญน้อยสุด)

แนวคิด

หาจำนวน เหรียญน้อยที่สุด ที่ทอนได้พอดีตามยอดที่กำหนด ต่างจาก Greedy ที่อาจให้คำตอบไม่ดีที่สุด กับบางชุดเหรียญ — DP รับประกันคำตอบที่ดีที่สุดเสมอ

dp[a] = จำนวนเหรียญน้อยสุดที่ทอนยอด a · กฎ: dp[a] = min(dp[a − c] + 1) สำหรับทุกเหรียญ c ≤ a

ความซับซ้อน: O(amount × จำนวนเหรียญ) · ตัวอย่าง: เหรียญ {1,3,4} ยอด 6 → Greedy ได้ 4+1+1=3 เหรียญ แต่ DP ได้ 3+3=2 เหรียญ
