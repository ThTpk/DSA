Bucket Sort (เรียงแบบถัง)

แนวคิด

Bucket Sort เหมาะกับข้อมูลที่กระจายสม่ำเสมอในช่วงที่รู้: แบ่งเป็น ถัง (bucket) ตามช่วงค่า โยนแต่ละตัวเข้าถังของมัน แล้ว เรียงในแต่ละถัง (เช่น insertion sort) สุดท้าย ต่อถังเรียงกัน ได้ผลลัพธ์

ความซับซ้อน: เฉลี่ย O(n + k) เมื่อข้อมูลกระจายดี · แย่สุด O(n²) ถ้าทุกตัวตกถังเดียว · ตัวอย่างนี้ค่า 0–99 แบ่ง 5 ถัง (ช่วงละ 20)

กำลังโยนเข้าถัง เรียงแล้ว/ผลลัพธ์

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: เรียงค่า [29, 25, 3, 49, 9, 37, 21, 43] (ช่วง 0–99) ที่กระจายค่อนข้างสม่ำเสมอ ด้วย Bucket Sort 5 ถัง (ช่วงละ 20)

แก้ด้วย Bucket Sort อย่างไร: โยนเข้าถังตามช่วง: ถัง0(0–19)=[3,9], ถัง1(20–39)=[29,25,37,21], ถัง2(40–59)=[49,43] · เรียงในแต่ละถัง (insertion sort): [3,9], [21,25,29,37], [43,49] · ต่อถังเรียงกัน → [3,9,21,25,29,37,43,49] · เฉลี่ย O(n + k) เมื่อข้อมูลกระจายดี (แต่ละถังมีของน้อย) · แย่สุด O(n²) ถ้าทุกตัวตกถังเดียว จึงเหมาะกับข้อมูลที่กระจายสม่ำเสมอและรู้ช่วงค่า

เจอได้ที่ไหน: เรียงเลขทศนิยมในช่วง [0,1) ที่กระจายสม่ำเสมอ, เรียงข้อมูลที่รู้การกระจาย (เช่นคะแนนสอบ), ใช้คู่กับ Counting/Radix · LeetCode: Maximum Gap, Contains Duplicate III (bucketing), Sort an Array
