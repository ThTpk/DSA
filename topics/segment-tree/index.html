Segment Tree (ต้นไม้ช่วง)

แนวคิด

Segment Tree ใช้ตอบคำถามแบบ ช่วง (range query) เช่น "ผลรวม/ค่าน้อยสุดของ A[l..r]" ได้เร็ว O(log n) และยัง แก้ค่า (update) ได้เร็วด้วย แต่ละโหนดเก็บผลสรุปของช่วงหนึ่ง รากครอบทั้งอาเรย์ ใบคือสมาชิกเดี่ยว โหนดพ่อ = รวมผลของลูกซ้าย+ขวา

ความซับซ้อน: build O(n) · query ช่วง O(log n) · update จุด O(log n) · ใช้ในงานที่มีทั้งถามช่วงและแก้ค่าบ่อยๆ

กำลังลงลึก/ซ้อนบางส่วน ช่วงที่นำมาตอบ นอกช่วง (ข้าม)
