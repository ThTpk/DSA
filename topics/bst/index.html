หมวด 2 · โครงสร้างข้อมูลแบบต้นไม้

Binary Search Tree (ต้นไม้ค้นหาแบบทวิภาค)

แนวคิด

BST คือต้นไม้ทวิภาคที่มีกติกา: ทุกโหนด ลูกซ้าย < พ่อ < ลูกขวา ทำให้การค้นหา/เพิ่ม/ลบ ทำได้โดยเปรียบเทียบแล้วเลือกไปทางซ้ายหรือขวา ตัดข้อมูลครึ่งหนึ่งทุกครั้ง (เหมือน Binary Search แต่บนโครงสร้างต้นไม้)

ความซับซ้อน: search/insert/delete เฉลี่ย O(log n), แย่สุด O(n) เมื่อต้นไม้เอียงเป็นเส้น (unbalanced) · เป็นการทำงานแบบ recursive
