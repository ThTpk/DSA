หมวด 1 · โครงสร้างข้อมูลเชิงเส้น

Array (อาเรย์)

แนวคิด

Array เก็บข้อมูลเรียงต่อกันในหน่วยความจำ เข้าถึงสมาชิกด้วย index ได้ทันที (O(1)) แต่การ แทรก/ลบ ตรงกลางต้อง เลื่อนสมาชิกที่เหลือ ทำให้ช้า O(n)

ความซับซ้อน: เข้าถึง index = O(1) · ค้นหา = O(n) · แทรก/ลบ = O(n) (เพราะต้องเลื่อน)

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: มีอุณหภูมิรายวัน [30, 32, 28, 35, 31] อยากหาค่าเฉลี่ย และดึงค่าของวันที่ 4 (index 3) มาแสดง

แก้ด้วย Array อย่างไร: เข้าถึง arr[3] = 35 ได้ทันทีด้วย index → O(1) เพราะข้อมูลเรียงต่อกันในหน่วยความจำ คำนวณที่อยู่ได้ตรง ๆ; วนรวมทุกค่าเพื่อหาเฉลี่ย → O(n) · เหมาะเมื่อรู้จำนวนสมาชิกแน่นอนและเน้นอ่านด้วยตำแหน่ง
