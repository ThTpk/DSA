Kadane's Algorithm (Max Subarray)

แนวคิด

หา ผลรวมของช่วงย่อยที่ติดกัน ที่มากที่สุด ในอาเรย์ (อาจมีค่าลบ) ด้วย DP เชิงเส้น O(n):

  • cur = ผลรวมช่วงที่ดีที่สุด ที่จบที่ตำแหน่งนี้
  • ถ้า cur + A[i] < A[i] → เริ่มช่วงใหม่ที่ A[i] (ทิ้งของเก่าที่ติดลบ)
  • best = max(best, cur) ตลอด

ความซับซ้อน: O(n) · ตัวเลขใต้กล่อง = ค่าในอาเรย์ · ช่วงเขียว = คำตอบ

ตำแหน่งปัจจุบัน ช่วง cur ช่วง best (คำตอบ)
