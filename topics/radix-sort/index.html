Radix Sort (การเรียงตามหลัก)

แนวคิด

Radix Sort เรียงจำนวนเต็มทีละหลัก จาก หลักหน่วย → สิบ → ร้อย (LSD) แต่ละรอบใช้การแจกเข้า ถัง 0–9 ตามเลขหลักนั้นแบบ เสถียร (stable) แล้วเก็บกลับตามลำดับถัง พอครบทุกหลักก็เรียงเสร็จ

ความซับซ้อน: O(d·(n + b)) เมื่อ d = จำนวนหลัก, b = ฐาน (10) · เร็วมากเมื่อเลขไม่ยาว · ใช้ Counting Sort เป็นตัวช่วยในแต่ละหลัก

หลักที่กำลังดู (sub = เลขหลักนั้น)
