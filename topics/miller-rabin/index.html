หมวด 7 · หัวข้อขั้นสูง

Miller-Rabin (ทดสอบจำนวนเฉพาะ)

แนวคิด

ตรวจว่า n เป็นจำนวนเฉพาะไหมแบบ เร็ว (ไม่ต้องหารทุกตัว) โดยเขียน n−1 = d·2ˢ (d เป็นเลขคี่) แล้วเลือก "พยาน (witness)" a มาทดสอบ: คำนวณ x = aᵈ mod n ถ้า x = 1 หรือ n−1 → ผ่าน ไม่งั้นยกกำลังสองซ้ำ ๆ หาว่าเจอ n−1 ไหม ถ้าไม่เจอเลย → n เป็นจำนวนประกอบแน่นอน

เป็นการทดสอบเชิงน่าจะเป็น: ถ้าผ่านทุกพยาน = "น่าจะเป็นจำนวนเฉพาะ" · ใช้พยานหลายตัวยิ่งมั่นใจ · จับ เลขคาร์ไมเคิล (เช่น 561) ได้ ต่างจากการทดสอบของแฟร์มา · O(k·log³n)
