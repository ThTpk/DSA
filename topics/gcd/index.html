Euclidean Algorithm (หา ห.ร.ม. + Extended)

แนวคิด

Euclid's Algorithm หาตัวหารร่วมมาก (GCD) จากข้อสังเกตว่า gcd(a, b) = gcd(b, a mod b) ทำซ้ำจน b = 0 → คำตอบคือ a ตัวสุดท้าย เร็วมาก (O(log min(a,b)))

Extended Euclidean หาเพิ่มว่าค่าสัมประสิทธิ์ x, y ที่ทำให้ a·x + b·y = gcd(a, b) (เอกลักษณ์ของเบซู) ใช้หา ตัวผกผันแบบมอดุลาร์ (modular inverse) ใน RSA และการเข้ารหัส
