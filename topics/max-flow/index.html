Maximum Flow (Edmonds-Karp)

แนวคิด

ส่ง "การไหล" จากต้นทาง S ไปปลายทาง T ผ่านท่อที่มี ความจุ (capacity) จำกัด ให้ได้ มากที่สุด · Ford-Fulkerson หา augmenting path (เส้นทางที่ยังเหลือความจุ) แล้วดันการไหลเพิ่มทีละเส้นทาง · Edmonds-Karp ใช้ BFS หาเส้นทางสั้นสุดเสมอ → O(V·E²)

ป้ายบนเส้น = flow / capacity · เมื่อไม่เหลือ augmenting path แล้ว = ได้การไหลสูงสุด · (ค่าการไหลสูงสุด = ความจุของ min-cut ตามทฤษฎี Max-Flow Min-Cut)

augmenting path (BFS) มีการไหลแล้ว

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: ระบบประปามีท่อเชื่อมจุดต่างๆ แต่ละท่อมีความจุน้ำต่อวินาทีจำกัด อยากรู้ว่าส่งน้ำจากแหล่ง S ไปถึงเมือง T ได้มากสุดกี่หน่วยต่อวินาที

แก้ด้วย Max Flow อย่างไร: หา augmenting path (เส้นทางที่ยังเหลือความจุ) แล้วดันน้ำเพิ่มเท่ากับคอขวดของเส้นทางนั้น พร้อมสร้าง residual edge ให้ย้อนได้ ทำซ้ำจนไม่มีเส้นทางเหลือ · Edmonds-Karp ใช้ BFS หาเส้นสั้นสุดทุกรอบ → O(V·E²) · ค่าการไหลสูงสุด = ความจุ min-cut (Max-Flow Min-Cut)
