Maximum Flow (Edmonds-Karp)

แนวคิด

ส่ง "การไหล" จากต้นทาง S ไปปลายทาง T ผ่านท่อที่มี ความจุ (capacity) จำกัด ให้ได้ มากที่สุด · Ford-Fulkerson หา augmenting path (เส้นทางที่ยังเหลือความจุ) แล้วดันการไหลเพิ่มทีละเส้นทาง · Edmonds-Karp ใช้ BFS หาเส้นทางสั้นสุดเสมอ → O(V·E²)

ป้ายบนเส้น = flow / capacity · เมื่อไม่เหลือ augmenting path แล้ว = ได้การไหลสูงสุด · (ค่าการไหลสูงสุด = ความจุของ min-cut ตามทฤษฎี Max-Flow Min-Cut)
