Floyd's Cycle Detection (เต่ากับกระต่าย)

แนวคิด

ตรวจว่า Linked List มี วงวน (cycle) ไหม ด้วยตัวชี้ 2 ตัว: slow (เต่า) เดินทีละ 1 ก้าว และ fast (กระต่าย) เดินทีละ 2 ก้าว ถ้ามีวงวน กระต่ายจะวิ่งวนมา เจอเต่า ในวง · ถ้าไม่มีวงวน กระต่ายจะถึง null ก่อน

ความซับซ้อน: เวลา O(n) · พื้นที่ O(1) (ไม่ต้องเก็บโหนดที่เคยไป) · ใช้ตรวจ memory leak, วนซ้ำในลำดับ ฯลฯ
