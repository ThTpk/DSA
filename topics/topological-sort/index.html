Topological Sort (เรียงลำดับเชิงทอพอโลยี)

แนวคิด

จัดลำดับโหนดในกราฟ มีทิศและไม่มีวงวน (DAG) ให้ทุกลูกศร ชี้ไปข้างหน้าเสมอ (ถ้า u→v แล้ว u ต้องมาก่อน v) ใช้กับงานที่มีลำดับก่อนหลัง เช่น ตารางเรียน, build dependencies, สูตรอาหาร

Kahn's algorithm: นับ in-degree (ลูกศรเข้า) ของแต่ละโหนด เอาตัวที่ in-degree=0 ออกก่อน แล้วลด in-degree ของเพื่อนบ้าน ทำซ้ำ · O(V + E) · ตัวเลขใต้โหนด = in-degree

in-degree=0 กำลังเอาออก จัดลำดับแล้ว
