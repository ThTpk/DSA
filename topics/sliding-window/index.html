Sliding Window (หน้าต่างเลื่อน)

แนวคิด

Sliding Window เป็นเทคนิคหาคำตอบของ "ช่วงต่อเนื่อง" อย่างมีประสิทธิภาพ ตัวอย่าง: หา ผลรวมมากสุดของช่วงยาว k — แทนที่จะบวกใหม่ทุกครั้ง (ช้า O(n·k)) เราเลื่อนหน้าต่าง เพิ่มตัวที่เข้า ลบตัวที่ออก ทำให้เหลือ O(n)

ความซับซ้อน: O(n) · ใช้กับปัญหาช่วงต่อเนื่องมากมาย (ทั้งขนาดคงที่และยืดหดได้)
