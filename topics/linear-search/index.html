หมวด 5 · การค้นหา

Linear Search (การค้นหาแบบเชิงเส้น)

แนวคิด

Linear Search ค้นหาแบบตรงไปตรงมา: ไล่ดูข้อมูล ทีละตัวจากซ้ายไปขวา เปรียบเทียบกับค่าที่ต้องการ ถ้าตรงก็คืนตำแหน่งทันที ถ้าดูครบทุกตัวแล้วไม่เจอก็แปลว่าไม่มี

ความซับซ้อน: เวลา O(n) แย่สุด/เฉลี่ย, O(1) ดีสุด (เจอตัวแรก) · พื้นที่ O(1) · ใช้ได้กับข้อมูล ที่ไม่ต้องเรียง
