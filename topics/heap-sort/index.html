Heap Sort (การเรียงด้วยฮีป)

แนวคิด

Heap Sort ใช้โครงสร้าง Max-Heap ช่วยเรียง 2 ขั้น:

  • สร้าง Max-Heap จากอาเรย์ (heapify จากล่างขึ้นบน) — ค่ามากสุดลอยไปที่ราก (index 0)
  • ดึงค่ามากสุด สลับไปไว้ท้าย แล้วลดขนาด heap ทำ sift-down ซ่อม heap ทำซ้ำจนหมด

ความซับซ้อน: O(n log n) ทุกกรณี · พื้นที่ O(1) (in-place) · ไม่ stable · มองอาเรย์เป็นต้นไม้: ลูกของ index i คือ 2i+1 และ 2i+2

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: เรียงอาเรย์ [4, 10, 3, 5, 1] จากน้อยไปมากด้วย Heap Sort

แก้ด้วย Heap Sort อย่างไร: สร้าง Max-Heap ก่อน (heapify) → [10,5,3,4,1] (10 อยู่ราก) · สลับราก 10 ไปท้ายสุด ลดขนาด heap แล้ว sift-down ซ่อม → ได้ค่ามากสุดเข้าที่ทีละตัว · ทำซ้ำ: ดึง 5, 4, 3 ... จนได้ [1,3,4,5,10] · O(n log n) ทุกกรณี และพื้นที่ O(1) (in-place) จึงเหมาะเมื่อต้องการเวลาแบบคงที่และประหยัดหน่วยความจำ (ไม่เหมือน merge ที่ใช้ O(n))
