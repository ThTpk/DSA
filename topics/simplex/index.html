Linear Programming / Simplex (เชิงเรขาคณิต)

แนวคิด

Linear Programming: หาค่ามากสุด/น้อยสุดของฟังก์ชันเชิงเส้น ภายใต้ ข้อจำกัดเชิงเส้น (อสมการ) เซตของจุดที่สอดคล้องทุกข้อจำกัด = พื้นที่เป็นไปได้ (feasible region) ซึ่งเป็นรูปหลายเหลี่ยมนูน

ทฤษฎีบทสำคัญ: ค่าเหมาะที่สุดอยู่ที่มุม (vertex) เสมอ · Simplex จึงเดินจากมุมหนึ่งไปมุมข้างเคียง ตามขอบที่ทำให้ objective ดีขึ้น เรื่อย ๆ จนไม่มีมุมข้างเคียงไหนดีกว่า = จุดเหมาะที่สุด

ตัวอย่าง: max 3x + 2y ภายใต้ x+y≤4, x+3y≤6, 3x+y≤9, x≥0, y≥0

พื้นที่เป็นไปได้ มุมปัจจุบัน จุดเหมาะที่สุด

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: โรงงานผลิตสินค้า 2 ชนิด มีเวลาเครื่องจักรและวัตถุดิบจำกัด จงหาจำนวนผลิตที่ให้กำไรสูงสุด

แก้ด้วย Simplex Method อย่างไร: เขียนเป็น linear program (เป้าหมาย + ข้อจำกัดเชิงเส้น) คำตอบที่ดีที่สุดอยู่ที่จุดยอดของ polytope ที่เป็นไปได้ Simplex เดินจากจุดยอดหนึ่งไปอีกจุดยอดที่ทำให้ค่าเป้าหมายดีขึ้นเรื่อย ๆ จนถึงจุดที่ดีที่สุด → กรณีจริงเร็วมาก (worst case exponential)

เจอได้ที่ไหน: วางแผนการผลิต/โลจิสติกส์ · จัดสรรทรัพยากร/พอร์ตลงทุน · diet problem · พื้นฐาน operations research และ optimization
