หมวด 5 · การค้นหา

Binary Search (การค้นหาแบบทวิภาค)

แนวคิด

Binary Search ใช้ได้กับข้อมูลที่ เรียงลำดับแล้ว เท่านั้น ดูที่ค่ากลาง (mid) ของช่วง ถ้าตรงก็จบ ถ้า target น้อยกว่าก็ ตัดครึ่งขวาทิ้ง ถ้ามากกว่าก็ ตัดครึ่งซ้ายทิ้ง ทำให้ขนาดที่ต้องค้นเหลือครึ่งเดียวทุกครั้ง

ความซับซ้อน: เวลา O(log n) — เร็วกว่า Linear Search มากเมื่อข้อมูลใหญ่ · พื้นที่ O(1) · ข้อแม้: ข้อมูลต้องเรียงก่อน

💡 ตัวชี้: lo = ขอบซ้าย · mid = ตัวที่เทียบ · hi = ขอบขวา
