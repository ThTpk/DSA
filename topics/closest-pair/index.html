Closest Pair of Points (คู่จุดที่ใกล้สุด)

แนวคิด

หาคู่จุดที่ระยะห่างน้อยสุดในระนาบ · วิธีตรงไปตรงมาเทียบทุกคู่ = O(n²) · Divide & Conquer ทำได้ O(n log n):

  • แบ่ง: เรียงจุดตามแกน x แล้วแบ่งครึ่งด้วยเส้นตั้ง
  • แก้: หาคู่ใกล้สุดในซ้าย (d_L) และขวา (d_R) ; d = min(d_L, d_R)
  • รวม: ตรวจเฉพาะจุดใน "แถบกลาง" กว้าง 2d รอบเส้นแบ่ง (เทียบกันแค่ไม่กี่จุด)

เส้นเขียว = คู่ที่ใกล้สุดที่พบ · เส้นประ = เส้นแบ่ง · แถบเทา = strip กลาง

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: มีจุด 100,000 จุดบนระนาบ จงหาคู่จุดที่อยู่ใกล้กันที่สุด

แก้ด้วย Closest Pair (divide & conquer) อย่างไร: เรียงจุดตามแกน x แบ่งครึ่งซ้าย-ขวา หาคู่ใกล้สุดในแต่ละฝั่งแบบ recursive แล้วตรวจเฉพาะจุดในแถบกลาง (strip) กว้าง 2δ → O(n log n) เร็วกว่าการเทียบทุกคู่ O(n²)

เจอได้ที่ไหน: ตรวจการชน (collision detection) ในเกม/หุ่นยนต์ · จัดกลุ่มข้อมูล (clustering) · ระบบควบคุมการจราจรทางอากาศหาเครื่องบินที่ใกล้กัน
