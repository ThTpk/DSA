Bloom Filter (ตัวกรองบลูม)

แนวคิด

Bloom Filter เช็กว่าสมาชิก "เคยถูกเพิ่มไหม" โดยใช้หน่วยความจำน้อยมาก ด้วย บิตอาเรย์ + hash function หลายตัว (k):

  • add(x): คำนวณ k ตำแหน่งจาก hash แล้วตั้งบิตเป็น 1
  • query(x): ถ้าบิตทั้ง k ตำแหน่งเป็น 1 → "อาจมี", ถ้ามีตัวใดเป็น 0 → "ไม่มีแน่นอน"

จุดเด่น: ประหยัดพื้นที่สุดๆ, เช็กเร็ว O(k) · ข้อแลก: อาจ "false positive" (บอกว่ามีทั้งที่ไม่มี) แต่ ไม่มี false negative · ใช้ใน cache, ฐานข้อมูล, เว็บเบราว์เซอร์
