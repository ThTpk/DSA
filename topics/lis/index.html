Longest Increasing Subsequence (LIS)

แนวคิด

LIS = ลำดับย่อยที่ เพิ่มขึ้น และ ยาวที่สุด (ไม่ต้องติดกัน) ในอาเรย์ เช่น [10, 9, 2, 5, 7, 18] มี LIS ยาว 4

DP: dp[i] = ความยาว LIS ที่ จบที่ตำแหน่ง i · dp[i] = 1 + max(dp[j]) สำหรับทุก j < i ที่ A[j] < A[i] (ถ้าไม่มีก็เป็น 1)

ความซับซ้อน: O(n²) แบบนี้ (มีวิธี O(n log n) ด้วย binary search) · ตัวเลขใต้กล่อง = dp[i]

ตัวที่กำลังคิด (i) ตัวที่เทียบ (j) LIS ที่ได้
