Z-algorithm (จับคู่สตริงแบบเชิงเส้น)

แนวคิด

Z-array ของสตริง s: Z[i] = ความยาวของสตริงย่อยที่เริ่มที่ i และ ตรงกับคำนำหน้า (prefix) ของ s คำนวณได้ใน O(n) ด้วยการจำหน้าต่าง [l, r] ที่แมตช์ล่าสุด เพื่อ ไม่ต้องเทียบซ้ำ

ใช้ค้นหาแพตเทิร์น: ต่อสตริง s = pattern + '#' + text ตำแหน่งที่ Z[i] = |pattern| คือจุดที่เจอแพตเทิร์น · เลขใต้ช่อง = ค่า Z · O(n + m)

ตำแหน่งปัจจุบัน i หน้าต่าง [l, r] เจอแพตเทิร์น (Z = |P|)

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: นับจำนวนครั้งที่แพตเทิร์น P ปรากฏใน T โดยต่อสตริงเป็น P + "$" + T

แก้ด้วย Z-Algorithm อย่างไร: คำนวณ Z-array (ความยาว prefix ที่ตรงกันที่ตำแหน่งแต่ละจุด) ในครั้งเดียวด้วย Z-box O(n) ตำแหน่งใดที่ Z[i] = |P| คือจุดที่ P ปรากฏใน T → string matching ใน O(n+m) ไม่ต้องสร้าง failure table แบบ KMP

เจอได้ที่ไหน: ค้นหาแพตเทิร์น · หาคาบ (period) ของสตริง · นับ distinct substring · LeetCode 28 (Find the Index of the First Occurrence)
