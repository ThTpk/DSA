Suffix Array (อาเรย์ส่วนต่อท้าย)

แนวคิด

Suffix Array คือลำดับของ ตำแหน่งเริ่มต้น ของ "ส่วนต่อท้าย (suffix)" ทั้งหมดของสตริง เมื่อ เรียงตามพจนานุกรม (lexicographic) · เป็นโครงสร้างที่เบากว่า suffix tree แต่ทรงพลัง

ใช้ค้นหาสตริงย่อยเร็ว (binary search บน SA), หา longest repeated substring, บีบอัดข้อมูล (BWT) · สร้างแบบเรียงตรง ๆ O(n² log n) (ตัวอย่างนี้) — ของจริงใช้ O(n log n) หรือ O(n)
