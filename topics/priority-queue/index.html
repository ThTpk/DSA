Priority Queue (คิวลำดับความสำคัญ)

แนวคิด

Priority Queue คือคิวที่ออกตาม ความสำคัญ ไม่ใช่ลำดับเข้า — ตัวที่สำคัญสุดออกก่อน (max-PQ) เช่น คิวงานเร่งด่วน, Dijkstra, Huffman

ตัวอย่างนี้แสดงเป็น อาเรย์ที่เรียงไว้ (มากไปน้อย) เพื่อให้เห็นภาพง่าย — enqueue = O(n), dequeue = O(1) · ของจริงนิยมใช้ Binary Heap → enqueue/dequeue = O(log n)
