Stack (สแตก)

แนวคิด

Stack ทำงานแบบ LIFO (Last In, First Out) — เข้าทีหลังออกก่อน เพิ่ม (push) และเอาออก (pop) ที่ ปลายบนสุด (top) เท่านั้น เหมือนกองจานที่วางซ้อนกัน

ความซับซ้อน: push / pop / peek = O(1) · ใช้กับ: ตรวจวงเล็บ, undo, เรียกฟังก์ชัน (call stack), DFS

📝 ตัวอย่างโจทย์ & การนำไปใช้

โจทย์: ตรวจว่าวงเล็บในสตริง "(a[b]{c})" จับคู่เปิด-ปิดถูกต้องไหม

แก้ด้วย Stack อย่างไร: เจอวงเล็บเปิด push, เจอวงเล็บปิด pop มาเทียบคู่; จบแล้ว stack ต้องว่าง · LIFO → O(n)

เจอได้ที่ไหน: คอมไพเลอร์, Undo/Redo, call stack · LeetCode: Valid Parentheses, Min Stack
