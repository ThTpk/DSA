หมวด 2 · โครงสร้างข้อมูลแบบต้นไม้

Trie (Prefix Tree / ต้นไม้คำนำหน้า)

แนวคิด

Trie คือต้นไม้ที่เก็บคำโดยใช้ ตัวอักษรเป็นเส้นทาง — คำที่มีคำนำหน้า (prefix) เหมือนกันจะใช้เส้นทางช่วงต้นร่วมกัน ทำให้ค้นหาคำหรือ prefix ได้เร็ว โหนดที่เป็น จุดจบคำ จะถูกทำเครื่องหมายไว้ (วงเขียว)

ความซับซ้อน: insert / search = O(L) เมื่อ L คือความยาวคำ (ไม่ขึ้นกับจำนวนคำที่เก็บ) · ใช้ทำ autocomplete, ตรวจคำสะกด, พจนานุกรม
