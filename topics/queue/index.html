Queue (คิว)

แนวคิด

Queue ทำงานแบบ FIFO (First In, First Out) — เข้าก่อนออกก่อน เหมือนคนต่อแถว เพิ่ม (enqueue) ที่ ท้าย (rear) และเอาออก (dequeue) ที่ หน้า (front)

ความซับซ้อน: enqueue / dequeue = O(1) · ใช้กับ: BFS, จัดคิวงาน, บัฟเฟอร์ · หมายเหตุ: ของจริงนิยมใช้ circular queue เลื่อนตัวชี้แทนการขยับสมาชิก
