Heap (Min-Heap)

แนวคิด

Heap คือต้นไม้ทวิภาคแบบสมบูรณ์ (complete binary tree) ที่มีกติกา Min-Heap: ทุกโหนดต้องมีค่าน้อยกว่าหรือเท่ากับลูกของมัน ดังนั้นค่าน้อยที่สุดอยู่ที่ราก เก็บได้ด้วยอาเรย์ (ลูกของ index i อยู่ที่ 2i+1 และ 2i+2)

  • insert: เพิ่มที่ท้าย แล้ว sift-up (ลอยขึ้นโดยสลับกับพ่อถ้าพ่อมากกว่า)
  • extract-min: เอารากออก ย้ายตัวสุดท้ายมาราก แล้ว sift-down (จมลงสลับกับลูกที่เล็กกว่า)

ความซับซ้อน: insert / extract-min = O(log n) · ดูค่าน้อยสุด = O(1) · ใช้ทำ Priority Queue และ Heap Sort
