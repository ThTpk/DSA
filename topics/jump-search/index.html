Jump Search (ค้นหาแบบกระโดด)

แนวคิด

Jump Search ใช้กับอาเรย์ที่ เรียงแล้ว โดยกระโดดทีละบล็อกขนาด ⌊√n⌋ จนเจอบล็อกที่ค่าปลายบล็อก ≥ เป้าหมาย แล้วค่อย ค้นเชิงเส้น ภายในบล็อกนั้น เร็วกว่า Linear Search แต่ช้ากว่า Binary Search

ความซับซ้อน: O(√n) · เหมาะเมื่อการ "กระโดดถอยหลัง" แพง (เช่นบางระบบจัดเก็บ)
